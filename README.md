# LPI Security Essentials

Security Essentials es una certificación de nivel fundamental del Linux Professional Institute (LPI) que introduce los conceptos esenciales de ciberseguridad, gestión de riesgos, criptografía, control de acceso, seguridad de redes y buenas prácticas para la protección de sistemas e información.

Esta certificación proporciona una base sólida para comprender los principios de seguridad informática y sirve como punto de partida para estudios más avanzados en ciberseguridad y administración segura de sistemas.

# Objetivo del repositorio

Documentar mi proceso de aprendizaje mediante notas, laboratorios prácticos y ejercicios relacionados con los temas cubiertos por la certificación Security Essentials.
