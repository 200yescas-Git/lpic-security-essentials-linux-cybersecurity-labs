# LPI Security Essentials

Este repositorio documenta mi proceso de estudio, prácticas y laboratorios para la certificación LPI Security Essentials.

Security Essentials es una certificación del Linux Professional Institute (LPI) enfocada en los fundamentos de la seguridad informática. Su objetivo es desarrollar una comprensión sólida de las amenazas comunes, mecanismos de protección, privacidad, criptografía y buenas prácticas de seguridad para usuarios y profesionales de TI.

---


## Objetivo del Repositorio

Este repositorio tiene como finalidad:

- Documentar mi proceso de aprendizaje.
- Organizar notas técnicas y documentación.
- Registrar laboratorios prácticos.
- Consolidar conocimientos fundamentales de ciberseguridad.
- Crear un portafolio técnico orientado a infraestructura, Linux y seguridad informática.

---

## Dominios de Conocimiento

### 1. Security Concepts

Fundamentos de seguridad informática y gestión de riesgos.

Temas principales:

- Confidencialidad, Integridad y Disponibilidad (CIA)
- No repudio
- Roles y actores de seguridad
- Tipos de amenazas
- Gestión de riesgos
- Vulnerabilidades
- CVE y CERT
- Zero-Day
- Pentesting
- Respuesta a incidentes
- Aspectos legales y regulatorios

### 2. Encryption

Conceptos esenciales de criptografía moderna.

Temas principales:

- Criptografía simétrica
- Criptografía asimétrica
- Funciones Hash
- PKI
- Certificados digitales
- X.509
- Autoridades certificadoras (CA)
- CSR
- RSA
- AES
- SHA-256
- Intercambio de claves
- Cifrado de datos en tránsito y reposo

### 3. Device and Storage Security

Protección de dispositivos y almacenamiento.

Temas principales:

- Seguridad física
- Hardware y componentes críticos
- IoT
- USB
- Bluetooth
- RFID
- Seguridad de aplicaciones
- Actualizaciones
- Malware
- Ransomware
- Troyanos
- Rootkits
- Copias de seguridad
- Estrategias de respaldo
- Seguridad en almacenamiento local y nube

### 4. Network and Service Security

Protección de redes y servicios.

Temas principales:

- Redes e Internet
- Servicios de red
- Protocolos seguros
- Firewalls
- Seguridad inalámbrica
- Seguridad de servicios
- Ataques comunes de red
- Protección de comunicaciones
- Acceso remoto seguro

### 5. Identity and Privacy

Protección de identidad digital y privacidad.

Temas principales:

- Autenticación
- Autorización
- MFA
- Gestión de contraseñas
- Protección de identidad digital
- Privacidad en línea
- Protección de datos personales
- Ingeniería social
- Phishing
- Gestión segura de cuentas


## Laboratorios

Los laboratorios de este repositorio tienen como objetivo reforzar los conceptos teóricos mediante ejercicios prácticos y escenarios reales.

Las prácticas incluyen:

- Configuración básica de seguridad.
- Gestión de usuarios y contraseñas.
- Verificación de integridad mediante hashes.
- Uso de certificados digitales.
- Análisis de malware y amenazas.
- Configuración de firewalls.
- Seguridad de redes.
- Protección de la identidad digital.
- Aplicación de buenas prácticas de seguridad.
