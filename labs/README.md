# Security Essentials Labs

Esta sección contiene los laboratorios prácticos desarrollados durante mi preparación para la certificación LPI Security Essentials.

## Objetivo

Reforzar los conceptos teóricos mediante ejercicios prácticos relacionados con seguridad informática, administración de sistemas Linux y buenas prácticas de protección de infraestructura.

## Entorno de Laboratorio

Para optimizar recursos y mantener consistencia entre las distintas certificaciones de mi ruta de aprendizaje, los laboratorios de este repositorio se realizan utilizando la misma máquina virtual Debian empleada en mis estudios de Linux Essentials, CompTIA Linux+ y LPIC.

### Plataforma utilizada

- VMware Workstation
- Debian GNU/Linux
- Acceso de usuario estándar y administrativo (sudo)
- Red configurada para prácticas de conectividad y servicios

## Áreas de práctica

Los laboratorios estarán enfocados en:

- Fundamentos de seguridad
- Criptografía
- Seguridad de dispositivos y almacenamiento
- Seguridad de redes y servicios
- Identidad, autenticación y privacidad
- Buenas prácticas de seguridad en Linux

## Estructura

Cada laboratorio incluirá:

- Objetivo
- Procedimiento
- Comandos utilizados
- Resultados obtenidos
- Evidencia cuando sea requerida
- Conclusiones
