# LPI Security Essentials

Security Essentials es una certificación de nivel fundamental del Linux Professional Institute (LPI) que introduce los conceptos esenciales de ciberseguridad, gestión de riesgos, criptografía, control de acceso, seguridad de redes y buenas prácticas para la protección de sistemas e información.

Esta certificación proporciona una base sólida para comprender los principios de seguridad informática y sirve como punto de partida para estudios más avanzados en ciberseguridad y administración segura de sistemas.

## Objetivo del repositorio

Documentar mi proceso de aprendizaje mediante notas, laboratorios prácticos y ejercicios relacionados con los temas cubiertos por la certificación Security Essentials.

El contenido incluye:

- Notas técnicas y documentación de estudio.
- Laboratorios prácticos.
- Evidencias de aprendizaje.
- Ejercicios de seguridad informática.
- Recursos complementarios para reforzar conocimientos.

## Dominios de conocimiento

### 1. Fundamentos de Seguridad

Comprensión de los principios fundamentales que permiten proteger información, sistemas y servicios.

#### Temas cubiertos

- Confidencialidad, Integridad y Disponibilidad (CIA)
- Gestión de riesgos
- Amenazas y vulnerabilidades
- Superficie de ataque
- Seguridad defensiva
- Principios de mínimo privilegio
- Defensa en profundidad

---

### 2. Criptografía

Introducción a los mecanismos utilizados para proteger información mediante técnicas criptográficas.

#### Temas cubiertos

- Cifrado simétrico
- Cifrado asimétrico
- Funciones hash
- Firmas digitales
- Certificados digitales
- Infraestructura de clave pública (PKI)
- Gestión de claves

---

### 3. Gestión de Identidad y Control de Acceso

Conceptos relacionados con la autenticación, autorización y administración de privilegios.

#### Temas cubiertos

- Identidad digital
- Autenticación
- Autorización
- Accounting
- MFA (Multi-Factor Authentication)
- Gestión de contraseñas
- Control de acceso basado en roles

---

### 4. Seguridad de Redes

Principios y tecnologías utilizadas para proteger la comunicación entre sistemas.

#### Temas cubiertos

- Firewalls
- Segmentación de redes
- VPN
- Protocolos seguros
- HTTPS
- TLS/SSL
- Seguridad inalámbrica
- Puertos y servicios comunes

---

### 5. Seguridad en Sistemas Linux

Aplicación de conceptos de seguridad dentro de entornos Linux.

#### Temas cubiertos

- Usuarios y grupos
- Permisos y propiedad de archivos
- sudo y privilegios administrativos
- Protección de servicios
- Hardening básico
- Monitoreo y registros del sistema
- Gestión de actualizaciones

---

### 6. Gestión de Riesgos y Buenas Prácticas

Prácticas orientadas a la reducción de riesgos y mejora continua de la postura de seguridad.

#### Temas cubiertos

- Gestión de vulnerabilidades
- Backups
- Recuperación ante incidentes
- Políticas de seguridad
- Concientización del usuario
- Protección de datos
- Continuidad del negocio

---

## Laboratorios

Los laboratorios incluidos en este repositorio están diseñados para reforzar los conceptos teóricos mediante actividades prácticas.

Algunos ejemplos incluyen:

- Gestión segura de usuarios y grupos.
- Configuración de permisos.
- Verificación de integridad mediante hashes.
- Implementación básica de firewalls.
- Uso de SSH seguro.
- Análisis de registros del sistema.
- Aplicación de principios de hardening.
